%1: Watchdog timeout.

Parameter:

%1......

Configuration path or name of object

Message Class:

3 -Error Minor

Reason:

The watchdog was not triggered by the application within the configured timeout.

Effect:

EtherNet/IP communication is not possible.

Solution:

Check if the application triggers the watchdog within the configured timeout or configure a longer timeout.
