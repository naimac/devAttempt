Messages

4000_615

Too little heap memory for cyclic logger (configured size: %1). Using default size (%2).

Parameter:

%1......

Configured size of the cyclic logger

%2......

Default size of the cyclic logger

Message Class:

3 -Error Minor

Reason:

Configuration error: Too little heap memory is available for the configured size of the cyclic logger.

Effect:

In case of errors diagnosis becomes more difficult.

Solution:

Provide information to your distributor: Create a Status Report and e-mail it to your distributor.
