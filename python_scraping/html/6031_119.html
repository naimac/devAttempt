UsedAddressBits mismatch largeModule=%1 FunctionID=%2 busNr=%3 moduleAddress=%4 expected=%5 read=%6

Parameter:

%1......

IOModule = 0, memory intensive module = 1

%2......

functionID in hex

%3......

BusNr

%4......

Moduleaddress (hexswitch) in hex

%5......

Number of used addressbits provided by driver

%6......

Number of used addressbits read from first found device

3 -Error Minor

Effect:
