Messages

6605_2

%1: Invalid configuration: Intersection of 'errorMask' (0x%2) and 'valueMask' (0x%3)

Parameter:

%1......

Configuration path or name of object

%2......

errorMask (hex)

%3......

valueMask (hex)

Message Class:

3 -Error Minor

Reason:

The same bit is set in 'errorMask' and in 'valueMask' at least once.

Effect:

The SSI-Device can not operate.

Solution:

Choose valid values for 'errorMask' and/or 'valueMask'
