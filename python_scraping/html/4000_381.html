Kin. %1: Excessive kinetic energy of workpiece against area %4 while executing motion command %2 (param. %3)

Parameter:

%1......

Name of robot

%2......

Name of segment

%3......

Segment parameter

%4......

Name of area

Message Class:

4 -Warning

Reason:

The robot would render the carried workpiece with an excessive kinetic energy.

Effect:

The robot and the motion program are stopped.

Solution:

Internal error. As a workaround, try programming a slower velocity.
