Messages

4000_610

Configured order of filter%1 (%2) does not correspond with the amount of coefficients (%3)

Parameter:

%1......

Filter number (1 or 2)

%2......

Configured filter order

%3......

Number of configured filter coefficients

Message Class:

4 -Warning

Reason:

Configuration error: The specified filter order does not match the number of filter coefficients. This is only checked with FIR-filters. Requirement: number of filter coefficients = filter order + 1.

Effect:

Filter is not activated.

Solution:

Correct filter configuration.
