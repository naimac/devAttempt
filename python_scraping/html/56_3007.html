Messages

56_3007

Size of shared memory ('%1': %3) differs from expected size (%2)

Parameter:

%1......

name of the shared memory

%2......

expected size

%3......

actual size of the shared memory

Message Class:

7 -

Reason:

The size of a shared memory, which is used by RC and PLC, differs from the expected size. The definition of the data structures in the RC doesn't match the one of the PLC.

Effect:

The PLC is stopped.

Solution:

Check versions of KAIRO and K_RC.library.
