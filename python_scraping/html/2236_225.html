Messages

2236_225

%1: Inconsistent data length of connection %2 (expected %3, configured %4).

Parameter:

%1......

Configuration path or name of the affected device

%2......

Connection number

%3......

expected connection length (calculated by the slave)

%4......

configured connection length (actualDataLength in SCI)

Message Class:

3 -Error Minor

Reason:

Configuration error. The data length of a connection does not match the data length calculated by the device.

Effect:

CP 3 cannot be reached. Initialization of bus is retried periodically.

Solution:

Correct the data length of the connection(actualDataLength) in the SCI.
