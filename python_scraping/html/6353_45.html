Messages

6353_45

%1: '%2': Mailbox '%3' is too small (act = %4, min = %5, section = '%6' / '%7').

Parameter:

%1......

Configuration path or name of object

%2......

Filename

%3......

Name of mailbox with invalid size

%4......

Actual size

%5......

Minimum size

%6......

Section of node

%7......

Subsection of node

Message Class:

3 -Error Minor

Reason:

The mailbox is too small in general or it is too small for at least one of the allowed protocols.

Effect:

The operation of the EtherCAT segment is not possible.

Solution:

Extend the size of the mailbox or allow less protocols.
