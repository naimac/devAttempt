4000_168

Incompatible tool %1 (ID %2) in spline command %3 (ID %4).

Parameter:

%1......

Name of incompatible tool

%2......

ID of incompatible tool

%3......

Name of spline command

%4......

ID of spline command

Message Class:

3 -Error Minor

Reason:

An incompatible tool has been used in a spline command. A spline consists of several single spline commands whose tool must not change.

Effect:

The spline cannot be appended to the path.

Solution:

You can either refrain from changing the tool or start a new spline.
