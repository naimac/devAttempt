Messages

6605_5

%1: The configured cycle time (%2 s) is not valid because comparator functionality is active.

Parameter:

%1......

Configuration path or name of object

%2......

Cycle time

Message Class:

3 -Error Minor

Reason:

Configured cycle time is different from the cycle time of the SSI parent node. This is not allowed if comparator functionality is active.

Effect:

The SSI device can not operate.

Solution:

Choose the same cycle time for the SSIDEVICE, SSI and SSI parent node or do not configure cycle time for SSI and SSIDEVICE at all (In this case the cycle time is derived from the SSI parent node).
