Messages

4000_685

Cannot transfer motion-buffer (ID: %1) to path (ID: %2) because appendFreeHdl=false!

Parameter:

%1......

Motion buffer ID

%2......

Execution path ID

Message Class:

4 -Warning

Reason:

Application error: Only one segment per update cycle may be transferred to the MCU. The boolean appendFree-flag informs you if the MCU is ready to accept new segments.

Effect:

The new motion commands are not accepted and are not executed.

Solution:

Check the application program for appendFree-flag.
