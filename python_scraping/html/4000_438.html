Messages

4000_438

Appending segment <%1> (ID %2) on path <%3> (ID %4) leads to a leap.

Parameter:

%1......

Motion command name

%2......

Kinematic ID

%3......

Path name

%4......

Path ID

Message Class:

3 -Error Minor

Reason:

Application error: A segment has been appended which does not connect to the previous path position. This causes a leap in the joints' position.

Effect:

Motion command is ignored. Program is stopped.

Solution:

Check the application program.
