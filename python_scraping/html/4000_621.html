Messages

4000_621

Kinematic %4: Invalid difference of configured minimum (%2) and maximum (%3) modulo positions of joint %1.

Parameter:

%1......

Name of joint

%2......

Configured minimum modulo position (moduloMinPos)

%3......

Configured maximum modulo position (moduloMaxPos)

%4......

Name of kinematic

Message Class:

3 -Error Minor

Reason:

The difference between the minimum and the maximum modulo positions is invalid. For rotatory joints the difference must be 360 degrees.

Effect:

The corresponding kinematic is not available.

Solution:

Check configuration data.
