Messages

2220_12026

%1: '%2' value conversion exceeds range (value: %3, factor: %4).

Parameter:

%1......

Configured name or path of device (drive)

%2......

Description of the value, e.g; set position, set velocity, ...

%3......

Value (of endpoint), which has caused range error

%4......

Used conversion factor (e.g; from configuration)

Message Class:

3 -Error Minor

Reason:

A value conversion (e.g; set value) from user units (endpoint value) to device units has exceeded the range, with the used conversion factor (e.g; for drives posUnitsToIncrement).

Effect:

Set value is not used (also no limit), changing to error state (if possible).

Solution:

Check conversion factor and set value.
