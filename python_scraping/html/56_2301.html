Messages

56_2301

%1: Set values from PLC are ignored by drive (%2).

Parameter:

%1......

name of the axis

%2......

value of parameter 0x28F1 read from drive

Message Class:

3 -Error Minor

Reason:

Parameter 0x28F1 is not set to 'DS402' (value: 2).

Effect:

Drive will not be operable.

Solution:

Set parameter 0x28F1 to 'DS402' (value: 2).
