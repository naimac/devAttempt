Messages

2236_5

%1: Data size of configured set values (MDT) too long (%2, maximum %3), default configuration used.

Parameter:

%1......

Hardware configuration path or name of device

%2......

Length of configured set values

%3......

Maximum allowed length of set values

Message Class:

4 -Warning

Reason:

Sum of length of all configured parameters in cyclic set values is greater than maximum allowed length (S-0-0186).

Effect:

Set value operation is performed with default configuration (a single set value depending on configured operating mode).

Solution:

Check configuration.
