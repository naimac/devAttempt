Messages

2220_12030

%1: Force temporary to value %2

Parameter:

%1......

Configured name or configuration-path of input or output.

%2......

New value for input or output

Message Class:

5 -Info

Reason:

Single write access to input or output variable.

Effect:

Input of output variable is set to force value one time. Force value remains on the varialbe until next access of application (for outputs) or IO-system (for inputs).
