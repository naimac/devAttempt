%1: no cyclic data received

Parameter:

%1......

configured name or hardware configuration path of the module

Message Class:

3 -Error Minor

Reason:

no new message since last update

Effect:

PDO, endoints and device will be set do error state.

Solution:

Check connection to the device, state of the device and parametration of the PDOs. If this error message occures repeated try to increase the cycle time and/or baud rate. A CAN bus record may help to identify the detailed error reason.
