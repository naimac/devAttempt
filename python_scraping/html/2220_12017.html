Messages

2220_12017

%1: Invalid data type for configuration entry '%2' (found: '%3', expected: '%4').

Parameter:

%1......

configuration path or name of the device

%2......

name of configuration entry

%3......

found data type

%4......

expected data type

Message Class:

3 -Error Minor

Reason:

An invalid data type has been used for an configuration entry.

Effect:

The device can not operate or the ressource can not be used.

Solution:

To prevent this message a valid data type has to be used for this configuration entry.
