Messages

4000_183

Incompatible external tool %1 in motion command %2.

Parameter:

%1......

Name of incompatible external tool

%2......

Name of motion command

Message Class:

3 -Error Minor

Reason:

An incompatible external tool has been used in a spline command or a parabola command. The same external tool as in the previous motion command must be used.

Effect:

The motion command cannot be appended to the path.

Solution:

You can either refrain from changing the external tool or use another type of motion command.
