Messages

2220_1021

%1: data address range (%2-%3) out of process data range (%4)

Parameter:

%1......

Configuration path of object.

%2......

Start of data area (bit)

%3......

End of data area (bit)

%4......

Border of data area (bit)

Message Class:

3 -Error Minor

Reason:

Sum of configured start address and data length is larger than whole process data length.

Effect:

Mapping of data of endpoint in process data not possible.

Solution:

Check configuration.
