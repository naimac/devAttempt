Messages

7261_6161

%1: ER 24-17 %2-%3 (0xFF08): Supply unit: DC link Overcurrent (error register %4)

Parameter:

%1......

Hardware configuration path or name of drive

%2......

Error location: 0: axis error; 1: system error

%3......

Corresponding device: 0, 1, 2: error on axis 0, 1, 2

%4......

Error register

Message Class:

3 -Error Minor

Reason:

Supply unit: DC link Overcurrent

Effect:

Drive can not be operated

Solution:

Try to fix problem at supply unit. Then, quit error on the axis module next to the supply unit.
