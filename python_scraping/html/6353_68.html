Messages

6353_68

%1: FMMUs %2 and %3 overlap

Parameter:

%1......

Configuration entry or name of EtherCAT slave

%2......

Index of first related FMMU

%3......

Index of FMMU overlapping the first

Message Class:

3 -Error Minor

Reason:

The logical address ranges of the two FMMUs overlap. If bit addresses are used, there must be at least 3 fill bytes in between. If these fill bytes do not exist, the message class is error. In any other case the message class is warning.

Effect:

Operation of the object is not possible.

Solution:

Adapt the engineering data file (ENI).
