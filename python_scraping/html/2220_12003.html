Messages

2220_12003

%1: Not able to operate with device firmware version %2. Update at least to %3.

Parameter:

%1......

configuration path or name of the device

%2......

current device firmware version.

%3......

suggested device firmware version

Message Class:

3 -Error Minor

Reason:

The control firmware needs to operate at least with the suggested device firmware version to ensure correct functionality.

Effect:

Device will be stopped.

Solution:

Update firmware to the suggested version.
