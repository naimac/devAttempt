Messages

2236_223

%1: Bustopology changed from '%2' to '%3'

Parameter:

%1......

Configuration path or name of bus

%2......

Shortcut of old bus topology ('N' = no link, 'L CON 0' = line CON 0, 'L CON 1' = line CON 1, 'DL' = double line, 'R' = ring, 'BR' = broken ring)

%3......

Shortcut of new bus topology (see parameter 2)

Message Class:

4 -Warning

Reason:

Bus topology changed due to connected or disconnected network cable.

Effect:

The cyclic data is transmitted via the new bus topology.

Solution:

If no change of the bus topology was desired, the network cables in use should be checked.
