Messages

6353_19

%1: Number of slaves in the EtherCAT segment (%2) is valid.

Parameter:

%1......

Configuration path or name of object

%2......

Number of slaves in the EtherCAT segment

Message Class:

5 -Info

Reason:

The number of slaves in the EtherCAT segment was invalid and has become valid again.

Effect:

The operation of the EtherCAT segment is possible

Solution:

The EtherCAT segment must have a valid number of slaves.
