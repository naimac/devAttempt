SingleJointLimiter can only be activated if torque-limiter is deactivated (KinID %1)

Parameter:

%1......

kinID

Message Class:

4 -Warning

Reason:

If the torque limiter is activated then it is not possible to activate the single joint limiter via McuShell.

Effect:

Single joint limiter will not be activated.

Solution:

Deactivate torque limiter if singel joint limiter is needed.
