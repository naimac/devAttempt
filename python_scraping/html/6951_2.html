Messages

6951_2

A new provider (prefix: '%1', TLP: '%2') collides with the existing provider (prefix: '%3', TLP: '%4').

Parameter:

%1......

Prefix of new provider

%2......

TLP of new provider

%3......

Prefix of existing provider

%4......

TLP of existing provider

Message Class:

3 -Error Minor

Reason:

Top level path (TLP) is already in use by the existing provider. It is not possible that the new provider uses this TLP too.

Effect:

The new provider cannot be added to Varaible Server.

Solution:

Please check provider configuration
