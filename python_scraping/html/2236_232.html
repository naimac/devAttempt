Messages

2236_232

%1: Error setting addresses: Actual device count (%2) different from configuration (%3)

Parameter:

%1......

Configuration path or name of bus

%2......

Number of connected devices on bus

%3......

Number of configured and not simulated devices

Message Class:

3 -Error Minor

Reason:

The number of connected devices is different from the number of configured devices. The number of configured devices does not include simulated devices.

Effect:

Automatic address setting is not executed. Phase bootup is stopped.

Solution:

Adjust configuration or connected devices.
