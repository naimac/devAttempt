Messages

4000_413

Kinematic %1 (joint %2): Referencing using reference pulse: No master value available.

Parameter:

%1......

Kinematic name

%2......

Joint name

Message Class:

4 -Warning

Reason:

The position of the drive cannot be set by using reference pulse as there is no master value available.

Effect:

The joint's position has not been set.

Solution:

The drive must be mastered first.
