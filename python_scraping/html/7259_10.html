Messages

7259_10

%1: Comparator group %2 contains outputs only.

Parameter:

%1......

Configuration path or name of the misconfigured object

%2......

ID of the outputs comparator group

Message Class:

4 -Warning

Reason:

There are no inputs in the outputs comparator group.

Effect:

The effected output comparators will never be triggered.

Solution:

Assign inputs to the comparator group or change the outputs group ID
