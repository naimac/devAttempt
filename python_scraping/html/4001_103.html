Messages

4001_103

'%1': Command buffer overflow (%2)

Parameter:

%1......

name of MFB object

%2......

length of command buffer

Message Class:

3 -Error Minor

Reason:

The command buffer of the MFB object is full.

Effect:

The command has not been accepted. At the moment no further commands can be applied to the MFB-object.

Solution:

To apply further commands one has to wait until the active command is terminated or the buffer length of the MFB object is increased by the configuration entry 'cmdBufferLength'.
