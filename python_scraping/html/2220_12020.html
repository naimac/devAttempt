Messages

2220_12020

%1: Value '%2' for configuration entry '%3' is no multiple of '%4'.

Parameter:

%1......

configuration path or name of the device

%2......

the invalid value configured for the configuration entry

%3......

name of configuration entry

%4......

step size defined for configuration entry

Message Class:

3 -Error Minor

Reason:

An invalid value has been configured for an configuration entry.

Effect:

The device can not operate or the ressource can not be used.

Solution:

A valid value has to be configured for the configuration entry.
