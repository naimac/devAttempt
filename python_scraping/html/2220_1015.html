Messages

2220_1015

%1: Endpoint '%2' already exists (%3)

Parameter:

%1......

Configuration path of endpoint or configured name of object and endpoint catalog name.

%2......

Name of endpoint

%3......

Configuration path of first endpoint with same name or configured name of first object and endpoint catalog name.

Message Class:

3 -Error Minor

Reason:

An endpoint with specified name already exists.

Effect:

Endpoint can not be used in system.

Solution:

Check configuration.
