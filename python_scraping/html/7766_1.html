Messages

7766_1

%1: %2-PDO data mapping for '%3' (0x%4:%5, len=%6) not found or incorrect.

Parameter:

%1......

Configuration path or name of object

%2......

Value "RX" indicates output data from control view, value "TX" indicates input data from control view

%3......

Text describing the kind of data in the mapping (e. g. DI, DO, DOSTAT, AI...)

%4......

Index of the CoE object whose mapping have not been found

%5......

Subindex of the CoE object whose mapping have not been found

%6......

Expected length of the CoE object whose mapping have not been found

Message Class:

3 -Error Minor

Reason:

The described CoE object have not been found in the PDO mapping in the engineering data file (ENI) or the mapping is not correctly byte aligned in the process data image.

Effect:

Operation of the object is not possible.

Solution:

Adapt the configuration file (cfg) or the the engineering data file (ENI).
