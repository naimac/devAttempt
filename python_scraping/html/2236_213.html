Messages

2236_213

%1: Slave valid bit already set when changing to phase %2.

Parameter:

%1......

Configuration path or name of bus or device (if available)

%2......

actual phase

Message Class:

3 -Error Minor

Reason:

Error in device. Behaviour not according to SERCOS specification. A device has set the 'slave valid'-bit.

Effect:

Phase startup cannot be continued

Solution:

Replace device causing this error
