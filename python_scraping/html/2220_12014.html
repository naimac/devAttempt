Messages

2220_12014

%1: Value '%2' for configuration entry '%3' is out of range [%4..%5].

Parameter:

%1......

configuration path or name of the device

%2......

The invalid value configured for the configuration entry

%3......

name of configuration entry

%4......

The smallest valid value

%5......

The largest valid value

Message Class:

3 -Error Minor

Reason:

The value configured for the configuration entry is out of range.

Effect:

The device can not operate.

Solution:

A valid value has to be configured for the configuration entry.
