Messages

2236_60

%1: Timeout on DRIVECOM state transition %2.

Parameter:

%1......

Hardware configuration path or name of drive

%2......

Concerned drivecom state transistion

Message Class:

3 -Error Minor

Reason:

Timeout occured on performing a drivecom state change during not-fulfilled conditions (e.g. halt of drives).

Effect:

Drive is set to drivecom state 'fault'.

Solution:

Leaving drivecom state 'operation enabled' drive must not move.
