%1: Device parameters not allowed (sampleTime = %2, baudRate = %3, valMask = %4, errMask = %5, databits = %6, datacode = %7, dataLsbFirst = %8)

Parameter:

%1......

Configuration path or name of object

%2......

sample time

%3......

baudRate

%4......

valueMask

%5......

errorMask

%6......

dataBits

%7......

dataCode

%8......

dataLsbFirst

Message Class:

3 -Error Minor

Reason:

The parameters configured are not allowed for the device.

Effect:

The SSI-Device can not operate.

Solution:

Use parameter values that are valid for the targeted SSI-Device.
