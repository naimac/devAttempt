Messages

2236_197

%1: Projected offset %2 of variable for IDN %3 does not match expected value %4.

Parameter:

%1......

Configuration path or name of affected device

%2......

projected offset [bits]

%3......

IDN of variable in projected process image

%4......

expected offset [bits]

Message Class:

3 -Error Minor

Reason:

Configuration error. The projected offset of a variable does not match the offset calculated from the attribute of the parameter.

Effect:

CP 3 cannot be reached. Bus initialization is not retried due to this configuration error.

Solution:

correct offset
