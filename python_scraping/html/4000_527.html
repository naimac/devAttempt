Messages

4000_527

Cannot activate/deactivate monitoring limit %1 while kinematic %2 is moving

Parameter:

%1......

Name of monitoring limit.

%2......

Name of kinematic.

Message Class:

3 -Error Minor

Reason:

Monitoring limits can only be changed during standstill, because dynamic monitoring is disabled.

Effect:

The robot stops. The limit will be changed at standstill.

Solution:

Only change monitoring limits when kinematic is not moving or else activate dynamic monitoring.
