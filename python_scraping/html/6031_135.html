KBus device address conflict, memory intensive module = %1, function ID = %2, bus number = %3, module address = %4

Parameter:

%1......

IO module = 0, memory intensive module = 1

%2......

Function ID in hex

%3......

Bus number

%4......

Module address (hex switch) in hex

Message Class:

3 -Error Minor

Reason:

Multiple modules of the same type with the same address (hex switch state) detected

Effect:

KBus not ready for operation

Solution:

Change hex switch state of modules and restart the control
