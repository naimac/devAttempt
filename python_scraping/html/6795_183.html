Messages

6795_183

%1: Execute SERCOS user command %2 failed (unknown command error code %3)

Parameter:

%1......

Hardware configuration path or name of device

%2......

Command number

%3......

Error code (hexadecimal)

Message Class:

5 -Info

Reason:

Execution of a Sercos-command triggered by applicatioin failed, unknown command error code

Effect:

Command not executed successfully

Solution:

Parameter values must be changed. The meaning of the error code can be found in the documentation of the device.
