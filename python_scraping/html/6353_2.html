Messages

6353_2

%1: At least one Ethernet frame was lost in the EtherCAT segment.

Parameter:

%1......

Configuration path or name of object

Message Class:

3 -Error Minor

Reason:

Error in the EtherCAT segment or hot connect of an EtherCAT slave.

Effect:

Data was lost.

Solution:

Check correct operation of slaves and cabling in the EtherCAT segment. If a hot connect was done, then ignore this message.
