Messages

3500_20003

Call of not linked Standard-AB-func (CREATE, DELETE, ACTIVATE, DEACTIVATE, RESET)

Parameter:

Message Class:

3 -Error Minor

Reason:

The IEC application calls a Standard-Func (AB, FB, Function), which does not exist in the firmware.

Effect:

The function cannot be used.

Solution:

Use a firmware version compatible with the used IEC-Std-Library.
