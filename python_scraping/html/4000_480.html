Forecast interpolator has fallen back (main int.: %1 %2 forecast int.: %3 %4)

Parameter:

%1......

Segment number of main run

%2......

Segment parameter of main run

%3......

Segment number of look-ahead

%4......

Segment parameter of look-ahead

Message Class:

5 -Info

Reason:

Error due to an internal problem in the system software: The look-ahead interpolator is behind the main interpolator.

Effect:

Errors in trigger prewarn time or excessive drive accelerations are possible.

Solution:

Provide information to your distributor: Create a Status Report and e-mail it to your distributor.
