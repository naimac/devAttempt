Messages

3700_99

%1 (%2/%3): Consistency-check failed

Parameter:

%1......

Filename

%2......

Line

%3......

Column

Message Class:

4 -Warning

Reason:

The data file (extension ttd) does contain inconsistent data, e.g. manually modified after the file was written.

Effect:

Application can not be compiled without errors.

Solution:

Deletion of the faulty data file. If the values in data file should be used despite the inconsistency, the CRC sum at the end of the file can be deleted.
