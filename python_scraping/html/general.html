General informations about error descriptions

This help file contains all error messages generated by the software components. These are classified by their internal component numbers. Each system error message contains the component number and the error number. The error description can be found by this number.

Syntax of a system error message:

[component number]_[error number]

Texts of system error messages consists of a fix text component and (optional) variable error texts (parameters). In this error description parameters are specified by %x, whereas x is a consecutive number. If a system error appears, these paramters are replaced with the according value automatically.

Example: parameter description

The text of the following error message includes some parameters (%x). All parameters are described in the list below.

2050_42 - Real value %1.%2 outside of range [%3..%4]

Parameter:

  • %1...... directory
  • %2...... name of entry
  • %3...... lowest possible value
  • %4...... highest possible value
