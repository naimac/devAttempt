Messages

4001_104

'%1': Invalid command sequence

Parameter:

%1......

name of MFB object

Message Class:

3 -Error Minor

Reason:

With certain objects certain sequences of commands are not allowed. On axes e.g. a phasing command, applied in buffered mode can't be followed by a further command in buffered mode.

Effect:

The command has not been accepted.

Solution:

not available
