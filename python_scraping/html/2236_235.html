Messages

2236_235

%1: Size %2 of IDN %3 not supported in cyclic data

Parameter:

%1......

configuration name of path of device

%2......

parameter length [bytes]

%3......

IDN with unsupported length

Message Class:

3 -Error Minor

Reason:

Configuration error. The actual length of a parameter is not supported in cyclic data, because it does not match the endpoint size.

Effect:

CP 3 cannot be reached. Bus initialization is not retried due to this configuration error.

Solution:

Do not configure cyclic IDN with unsupported length.
