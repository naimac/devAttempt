Messages

4000_302

Robot type (%1 joints) incompatible with kinematic %2 (%3 joints configured)

Parameter:

%1......

Joint number of robot type

%2......

Kinematic name

%3......

Number of kinematic joints

Message Class:

4 -Warning

Reason:

Configuration error: The configured type of robot requires a certain number of joints. However, a different number of joints has been configured.

Effect:

Kinematic is not available.

Solution:

Check the configured number of joints.
