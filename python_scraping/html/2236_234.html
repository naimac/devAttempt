%1: Device order mismatch

Parameter:

%1......

Configuration path or name of bus

Message Class:

3 -Error Minor

Reason:

Electronic type label and addresses of devices on bus match the configuration, but the order of the devices does not match.

Effect:

Automatic address setting is not executed. Phase bootup is continued.

Solution:

Adjust projection or topology such that the order of the devices matches.
