Messages

6242_5

KMEM: Cannot allocate specified amount of bytes (%1 bytes)

Parameter:

%1......

Number of bytes specified in IEC.Memory.

Message Class:

3 -Error Minor

Reason:

Not enough dynamic heap memory available to initialize KMEM.

Effect:

All KMEM-Functions will return error code "Status_NoMem".

Solution:

Check configuration entry "IEC.Memory".
