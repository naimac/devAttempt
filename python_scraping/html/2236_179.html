%1: Drive state can't be changed while Statusreport is active

Parameter:

%1......

Configured name or Hardware configuration path of the drive

Message Class:

3 -Error Minor

Reason:

The application tried to switch the drive to "Drive halt" or "Drive enabled" while a statusreport was active.

Effect:

The drive state can't be changed

Solution:

Wait until the statusreport has finished
