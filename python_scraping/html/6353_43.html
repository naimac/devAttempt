%1: %2: Auto increment address not allowed (section = '%3' / '%4').

Parameter:

%1......

Configuration path or name of object

%2......

Filename

%3......

Section of node

%4......

Subsection of node

Message Class:

3 -Error Minor

Reason:

Auto increment addressing is used during a state transition, which does not allow this addressing.

Effect:

The operation of the EtherCAT segment is not possible.

Solution:

Do not use auto increment addressing.
