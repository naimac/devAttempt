7458_1

%1: %2Fatal error 0x%3

Parameter:

%1......

Configuration path or name of (S)MM36x

%2......

Name of subdevice (optional)

%3......

Error code (hexadecimal)

Message Class:

3 -Error Minor

Reason:

The (S)MM36x reported a fatal error. For a detailed reason see the error code description in the documentation of the (S)MM36x.

Effect:

Outputs are deactivated.

Solution:

A fatal error can only be resolved by restarting the (S)MM36x (Power OFF/ON).
