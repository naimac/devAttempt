Messages

2236_169

%1: Drive not ready for power on (SERCOS drive state 0x%2).

Parameter:

%1......

Hardware configuration path or name of drive

%2......

Value of sercos stateword S-0-0135

Message Class:

3 -Error Minor

Reason:

In drivecom state 'ready to switch on' drive signalled in sercos stateword S-0-0135 the state 'drive not ready for power on' (bits 14 and 15 are 0).

Effect:

Drive is set to drivecom state 'fault'.

Solution:

Check drive and its power supply.
