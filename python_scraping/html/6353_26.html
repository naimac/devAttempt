%1: The cyclic commands have too many different cycle times (max = %2).

Parameter:

%1......

Configuration path or name of object

%2......

Maximum number of different cycle times

Message Class:

3 -Error Minor

Reason:

The cyclic commands are sent with too many different cycle times.

Effect:

The operation of the EtherCAT segment is not possible.

Solution:

Reduce the number of different cycle times of the cyclic commands.
