Messages

4000_522

Sum of wrist joints exceeds limit; value: %1, limit: %2, motion command (%3) %4, parameter %5

Parameter:

%1......

Current sum of wrist joints

%2......

Limit for sum of wrist joints

%3......

Position number of segment in the path

%4......

Segment name

%5......

Segment parameter (0..1)

Message Class:

3 -Error Minor

Reason:

Application error. A violation of the sum of wrist joints has been detected. The path must be modified.

Effect:

Robot and motion program are stopped.

Solution:

Increase sum of wrist joints limit in the configuration or modify program.
