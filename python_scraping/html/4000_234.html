Messages

4000_234

Overlap position %1 could not be calculated. Motion command %2 (ID %3).

Parameter:

%1......

Overlap position (1: start, 2: end)

%2......

Segment name

%3......

Segment ID

Message Class:

4 -Warning

Reason:

Error due to an internal problem in the system software.

Effect:

The robot stops. Further motion commands will not be accepted. Probably a series of other error messages will be issued which all relate to this error.

Solution:

Provide information to your distributor: Create a Status Report and e-mail it to your distributor.
