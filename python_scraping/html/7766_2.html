Messages

7766_2

%1: SYNC manager configuration error.

Parameter:

%1......

Configuration path or name of object.

Message Class:

3 -Error Minor

Reason:

An emergency error message for SYNC manager configuration has been triggered.

Effect:

The device can not perform the requested state transition. An entry in the diagnosis history is made (object 0x10F3).

Solution:

Correct EtherCAT configuration (ENI, cfg).
