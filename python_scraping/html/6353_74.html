Messages

6353_74

%1: Configured FoE file transfer failed (file '%2', state %3, cause %4, abort code 0x%5).

Parameter:

%1......

Configuration path or name of object

%2......

Name of file to transfer (without path)

%3......

State (possible values: 0 .. Error requesting BOOTSTRAP, 1 .. Error starting FoE request, 2 .. Error at FoE transfer)

%4......

Detailled error cause (possible values: 1 .. 'general error', 2 .. 'invalid parameter', 3 .. 'invalid file', 4 .. 'no FoE mailbox', 5 .. 'aborted', 6 .. 'timeout')

%5......

Abort code

Message Class:

3 -Error Minor

Reason:

The configured file transfer via FoE failed.

Effect:

The file was not transferred to the slave.

Solution:

Correct file content or file name.
