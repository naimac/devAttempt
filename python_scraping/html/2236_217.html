%1: Input values are not valid.

Parameter:

%1......

Configuration path or name of device

Message Class:

4 -Warning

Reason:

Error in device. The device has not set the 'inputs valid' bit in the status word (S-0-1500.0.2).

Effect:

The inputs have set an error; The input values are not valid

Solution:

Contact device manufacturer
