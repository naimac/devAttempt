%1: Robot is already reserved from another program, action '%2'

Parameter:

%1......

Name of interface instance.

%2......

Failed action, e.g. prog start.

Message Class:

8 -

Reason:

The robot is already reserved by another program, it is not possible to start a second program while the robot is reserved.

Effect:

Action cannot be executed.

Solution:

Unload active program before a new program is started.
