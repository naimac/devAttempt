Messages

4000_670

McuReleaseDbObj: object with ID %1 has already been released

Parameter:

%1......

ID of database object

Message Class:

5 -Info

Reason:

Application error: A database object can either be of type eMcuPermanent or eMcuTemp. If it is permanent this message occurs if the object has been released more often than once. If it is temporary this message occurs every time the object is released, because there is no need to release it.

Effect:

Unspecific effect.

Solution:

Check the application program.
