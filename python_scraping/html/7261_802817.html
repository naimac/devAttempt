Messages

7261_802817

%1: Execute SERCOS command %2 failed (%3C4001 Error during safe homing procedure, user = %4)

Parameter:

%1......

Hardware configuration path or name of drive

%2......

Command number

%3......

optional user comment

%4......

Calling context. If this parameter 1, the command was triggered by the application, else by the initialization (sercos phase bootup) or error acknowledgment.

Message Class:

3 -Error Minor

Reason:

Execution of a Sercos-command failed. For details see command error code.

Effect:

Command was not executed successfully.

Solution:

Parameter values must be changed. The cause can be narrowed by the command error code.
