Messages

6983_9

%1: RX-FIFO overrun (SI=%2, source=%3).

Parameter:

%1......

Configuration path or name of object

%2......

Number of SI-channel on SM2x0

%3......

Source of overrun (UART or Modul)

Message Class:

3 -Error Minor

Reason:

A recieving FIFO overrun has occured.

Effect:

Data on serial input has been lost

Solution:

Read data from serial input more often in your application and/or configure a larger input buffer in your ENI and/or use a faster cycle time for ECAT
