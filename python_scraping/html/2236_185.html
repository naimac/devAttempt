Messages

2236_185

%1: Too much data configured for bus cycletime (%2 s).

Parameter:

%1......

Configuration path or name of bus

%2......

bus cycle time in s

Message Class:

3 -Error Minor

Reason:

The configured bus cycletime does not suffice to transfer all data.

Effect:

CP 3 cannot be reached. Initialization of bus is retried via QuitError-Task.

Solution:

Check configuration. Configure less cyclic data or a greater cycle time.
