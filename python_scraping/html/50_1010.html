Messages

50_1010

Control Authority not granted to handterminal '%2' (IP: %1) since no handterminal is configured.

Parameter:

%1......

IP-address of connected handterminal.

%2......

Type of connected handterminal.

Message Class:

7 -

Reason:

Control Authority was requested for a handterminal while no handterminal is present in PLC configuration.

Effect:

Handterminal can't be used to move robot.

Solution:

Add a handterminal to PLC configuration.
