Messages

4000_950

Transfer of motion-buffer (ID: %1) has been interrupted - error in <McuAppendToPath>

Parameter:

%1......

MotionBuffer ID

Message Class:

4 -Warning

Reason:

Application error: Error in McuAppendToPath while transferring a motion buffer.

Effect:

The motion commands stored in the motion buffer may have been incompletely transferred - hence they will not be executed.

Solution:

Check previous error messages for hints to the problem.
