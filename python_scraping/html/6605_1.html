Messages

6605_1

%1: Invalid configuration: Sample time (%2 sec) too short for baudrate (%3 kBaud) and/or number of databits (%4).

Parameter:

%1......

Configuration path or name of object

%2......

sampleTime [seconds]

%3......

baudrate (kBaud)

%4......

Number of databits (of SSIDEVICE)

Message Class:

3 -Error Minor

Reason:

The configured sample time is too short for transmitting the configured number of databits in time (with the baudrate configured).

Effect:

The SSI-Bus can not operate.

Solution:

Use greater values for sample time and/or baudrate.
