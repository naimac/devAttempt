%1: Used polling time (%2 s) may lead to data losses (minium %3 s).

Parameter:

%1......

name of hardware-configurationpath of Sio

%2......

actual polling time

%3......

needed minimal polling time

Message Class:

4 -Warning

Reason:

With the set polling time not all baudrates can be handled sufficiently fast.

Effect:

Data loss is possible

Solution:

Decrease polling time (f.e. via system tick).
