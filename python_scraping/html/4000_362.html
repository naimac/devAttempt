Messages

4000_362

Kin. %1 is leaving work area %4 with guard point %6 of %5%7 while executing motion command %2 (param. %3)

Parameter:

%1......

Kinematic name

%2......

Segment name

%3......

Segment parameter

%4......

Name of the blocked area

%5......

Name of guard points object

%6......

Name of guard point

%7......

Name of object containing the guard point, e.g a tool, if existent

Message Class:

3 -Error Minor

Reason:

User error: Robot movement with workspace violation (working area)

Effect:

Robot and motion program are stopped.

Solution:

Change the motion program with respect to the direction of the movement.
