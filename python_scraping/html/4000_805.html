TrackingLogger status: Buffer overflow of tracking logger. Configured size: %1

Parameter:

%1......

Buffer size

Message Class:

5 -Info

Reason:

A tracking logger reports a buffer overflow. Logged data is lost.

Effect:

No effect, but diagnosis will be more difficult in case of errors.

Solution:

Increase tracking logger size in configuration.
