Messages

4000_210

TargetMode needed for a defined motion out of a singular or undefined mode position (%1 (ID %2)).

Parameter:

%1......

Kinematic name

%2......

Kinematic ID

Message Class:

4 -Warning

Reason:

Application error: Target mode for moving out of the wrist joint singularity has not been specified.

Effect:

You cannot move out of a singularity without specifying a target mode. Program or motion command is not executed.

Solution:

Submit the target mode for the affected motion command.
