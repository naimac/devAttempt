Messages

7259_11

%1: In KTY mode comparator functionality is not supported.

Parameter:

%1......

Configuration path or name of endpoint.

Message Class:

4 -Warning

Reason:

The analog input is configured for KTY mode. Hence comparator functionality is not supported for this input.

Effect:

The comparator of this endpoint can not be activated or prametrized.

Solution:

To use comparator functionality for this endpoint an other operating mode has to be configured.
