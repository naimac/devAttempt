Kinematic %1 has not been registered with the MCU execution unit

Parameter:

%1......

Kinematic name

Message Class:

4 -Warning

Reason:

It has been tried to use a kinematic not known to the MCU. Possibly, there is an error in the configuration data, such that the kinematic has not been created.

Effect:

Kinematic is not available. Programs cannot be started.

Solution:

Look at additional messages.
