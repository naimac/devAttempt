%1: Memory area %2.%3 - %4.%5 is not received completely.

Parameter:

%1......

Configuration path or name of object or slave address

%2......

Start byte address

%3......

Start bit address

%4......

Stop byte address

%5......

Stop bit address (exclusive)

Message Class:

3 -Error Minor

Reason:

No cyclic command reads this memory area completely.

Effect:

The operation of the EtherCAT segment is restricted.

Solution:

Add cyclic command or correct memory area.
