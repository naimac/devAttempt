Messages

7458_2

%1: %2Alarm 0x%3

Parameter:

%1......

Configuration path or name of (S)MM36x

%2......

Name of subdevice (optional)

%3......

Error code (hexadecimal)

Message Class:

3 -Error Minor

Reason:

The (S)MM36x reported an alarm. This alarm can be quit by the safety application. For a de-tailed reason see the error code description in the documentation of the (S)MM36x.

Effect:

Outputs are deactivated.

Solution:

Resolve the cause of the alarm, then quit the error in the safety application. For details see the error code description in the documentation of the (S)MM36x.
