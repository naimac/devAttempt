Messages

2236_173

%1: Auxiliary set value '%2' exceeds range (value: %3, factor: %4).

Parameter:

%1......

Hardware configuration path or name of drive

%2......

Number of addional set-value (index in aux-array)

%3......

Endpoint value, that caused the range overflow

%4......

Used transformation factor (read from parameter attribute)

Message Class:

4 -Warning

Reason:

A addtional set-value transformation from user units (endpoint value) to drive units using the transformation factor read from drive (parameter attribute) resulted a range overflow.

Effect:

Addional set-value is not transmitted to drive.

Solution:

Check addional set-values.
