Messages

4000_178

Overlap skipped because mode of overlap start position (%2) is different to target mode (%3) of segment <%4>; kinematic <%5> - <%1>.

Parameter:

%1......

Look-ahead or main run

%2......

Mode at overlap start

%3......

Target mode

%4......

Segment name

%5......

Kinematic name

Message Class:

4 -Warning

Reason:

Overlapping is impossible due to different start and target mode.

Effect:

No overlapping between segments.

Solution:

Start overlapping if start and target mode are the same.
