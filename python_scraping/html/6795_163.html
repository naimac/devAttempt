Messages

6795_163

%1: Execute SERCOS command %2 failed (%3, %4)

Parameter:

%1......

Hardware configuration path or name of device

%2......

Command number

%3......

Error code

%4......

Additional information

Message Class:

3 -Error Minor

Reason:

Execution of a command during initialization (sercos phase bootup) or error acknowledgment returned an error.

Effect:

Communcation between controller and device is disturbed.

Solution:

Cause can be narrowed by error code (3rd parameter). A specific SERCOS error code is shown in the trace if the traceWord is activated.
