Messages

2236_195

%1: Projected data length %2 of connection %3 does not match expected value %4.

Parameter:

%1......

Configuration path or name of affected device

%2......

projected connection length

%3......

connection number

%4......

expected connection length

Message Class:

3 -Error Minor

Reason:

Configuration error. The projected data length of a connection does not match the length calculated from the attributes of the parameters.

Effect:

CP 3 cannot be reached. Bus initialization is not retried due to this configuration error.

Solution:

correct projected connection length
