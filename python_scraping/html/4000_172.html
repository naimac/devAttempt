%5: Mode (%3) of target position <%7> of %6segment <%4> differs in flag <%8> from mode (%2) at start (<%1>).

Parameter:

%1......

Source of defect (Look-ahead or main run)

%2......

Mode at the start of the motion

%3......

Calculated mode for the target position

%4......

Segment name

%5......

Kinematic name

%6......

Interpolation type

%7......

Name of target position

%8......

Name of problematic mode flag

Message Class:

3 -Error Minor

Reason:

User error: The target mode in the motion program does not match the calculated target mode. The target position is not reachable.

Effect:

Motion program and robot are stopped.

Solution:

Check the target position in the motion program at the specified position.
