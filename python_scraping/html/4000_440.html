Stop of micro interpolator of kinematic <%1> not possible because the micro interpolator is not configured.

Parameter:

%1......

Kinematic name

Message Class:

3 -Error Minor

Reason:

Application error: The command McuStopMicroIP was used although no micro interpolator is configured.

Effect:

Instead of a micro interpolator stop a stop joints of the main interpolator is used.

Solution:

Check application program or configuration.
