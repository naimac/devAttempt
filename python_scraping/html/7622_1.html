%1: Message occurred (%2/%3)

Parameter:

Configuration path or name of device

Component number of message, 0 if number was not transmitted

Message number of message, 0 if number was not transmitted

5 -Info

Reason:

A message occured on specified device.

Effect:

The message on the device could be used for finding reasons for errors.

Solution:

For more detailed analysis a Statusreport can be created at the device.
