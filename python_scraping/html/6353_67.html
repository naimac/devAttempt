6353_67

%1: Slave is not of type '%2'

Parameter:

%1......

Configuration path or name of object

%2......

Configured slave type

Message Class:

3 -Error Minor

Reason:

The configured EtherCAT slave type does not match with the type from the engineering data file (ENI)

Effect:

Operation of the object is not possible.

Solution:

Adapt the configuration file (cfg) or the the engineering data file (ENI).
