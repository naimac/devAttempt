Messages

7525_6

%1: Invalid sync control 0x%2 in DC mode %3.

Parameter:

%1......

Configuration path or name of object

%2......

Value of Sync control register 0x980:0x981 of EtherCAT Slave

%3......

Configured DC mode (0=DC off, 1=DC Sync0, 2=DC Sync1)

Message Class:

3 -Error Minor

Reason:

DC configuration in EtherCAT master doesn't match with DC coniguration in EtherCAT slave.

Effect:

EtherCAT operation is not possible, EtherCAT state bootup is aborted with error.

Solution:

DC configuration in EtherCAT master and slave has to be checked.
