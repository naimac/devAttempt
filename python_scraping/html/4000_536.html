Messages

4000_536

End position of segment %1 exceeds limit of joint %2 by %3 %% (value: %4, limit: %5) of kinematic %6

Parameter:

%1......

Name of segment

%2......

Affected joint

%3......

Position excess in percent

%4......

Joint position

%5......

Violated limit of joint position

%6......

Name of kinematic

Message Class:

7 -

Reason:

The target position has been checked and a limit switch violation has been detected.

Effect:

The robot is stopped. The erroneous target position will not be used.

Solution:

Use a target position whose joint position lies within the allowed range.
