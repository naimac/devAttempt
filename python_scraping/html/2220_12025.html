%1: The text for configuration entry '%2' has an invalid length (min %3, max %4 characters).

Parameter:

%1......

configuration path or name of the device

%2......

name of configuration entry

%3......

minimum number of characters

%4......

maximum number of characters

Message Class:

3 -Error Minor

Reason:

The text configured for a configuration entry is too short or too long.

Effect:

The device can not operate or the ressource can not be used.

Solution:

A valid value has to be configured for the configuration entry.
