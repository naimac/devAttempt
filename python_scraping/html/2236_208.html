Messages

2236_208

%1: Value for configuration entry '%2' conflicts with one or more values in the engineering data file.

Parameter:

%1......

Configuration path or name of the node containing the configuration entry

%2......

name of configuration entry

Message Class:

3 -Error Minor

Reason:

Inconsistency between configuration and projection. Value for a configuration entry conflicts with one or more a value in the engineering data file.

Effect:

Device cannot be operated.

Solution:

Resolve error in engineering data file or in configuration file.
