Messages

6605_6

%1: 'SSI' and 'SSIDEVICE' can only operate in simulation mode if the parent node of 'SSI' operates in simulation mode too.

Parameter:

%1......

Configuration path or name of object

Message Class:

3 -Error Minor

Reason:

The 'SSI' or 'SSIDEVICE' node has been configured as 'simulated' but the parent node of 'SSI' is not configuerd as 'simulated'.

Effect:

The SSI device can not operate and can not be simulated.

Solution:

Configure the parent node of 'SSI' as simulated.
