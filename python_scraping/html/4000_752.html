Messages

4000_752

Wrong number of joints (%1 instead of %2) specified for dynamic robot model of kinematic <%3>

Parameter:

%1......

Wrong number of joints

%2......

Correct number of joints

%3......

Kinematic ID

Message Class:

4 -Warning

Reason:

The number of joints of the provided dynamic robot data (main joints + chained auxiliary joints + hidden joints) does not match the number of joints of the kinematic.

Effect:

The dynamic model will not be calculated.

Solution:

Check the configuration data of the torque model.
