Messages

6983_6

%1: Module mismatch, force module configuration failed.

Parameter:

%1......

Configuration path or name of object

Message Class:

3 -Error Minor

Reason:

The modules configured in the ESI do not match the modules found on the BL27x and could not be forced to the desired module mapping or the module configuration could not be read from the BL27x at all

Effect:

Operation of the object is not possible.

Solution:

Adapt the engineering data file (ENI). As a hint a list of expected/actual modules is listed in the trace if the traceWord is activated.
