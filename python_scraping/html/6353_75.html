Messages

6353_75

%1: Configured FoE file transfer failed (file '%2', state %3, AL Status code 0x%4, error code 0x%5).

Parameter:

%1......

Configuration path or name of object

%2......

Name of file to transfer (without path)

%3......

State (possible values: 0 .. Error when reading AL status code, 1 .. Timeout when polling AL status code, 2 .. FoE transfer error, 3 .. starvation timeout)

%4......

AL status code

%5......

Error code (see documentation of slave, 0xFFFFFFFF if not available)

Message Class:

3 -Error Minor

Reason:

The configured file transfer via FoE failed.

Effect:

The file was not transferred to the slave.

Solution:

Correct file content or file name. Execute slave specific resolution depending on error code.
