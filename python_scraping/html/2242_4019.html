Messages

2242_4019

%1: Bad quality (bus passive count: %2, bus warning count %3)

Parameter:

%1......

configuration path or name of the can channel

%2......

counter of occurred bus passive events

%3......

counter of occurred bus warning events

Message Class:

4 -Warning

Reason:

Bad bus quality, if at least one bus-passive or bus-warning event occurred.

Effect:

If the reason for this message is an interrupted connection or a missing partner to quit CAN-messages, all configured devices and endpoints of the bus are set to errorstate. Bus communication is recreated with Quit-Error-task

Solution:

check quality of CAN-connections (contact, shielding) and bus termination.
