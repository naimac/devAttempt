Messages

4000_951

<McuTerminateMotionBuffer> not possible - transfer of motion-buffer (ID: %1) is active

Parameter:

%1......

MotionBuffer ID

Message Class:

4 -Warning

Reason:

Application error: The command McuTerminateMotionBuffer must not be called in case transferring the motion buffer is currently active.

Effect:

Transfer is not finished.

Solution:

Send command McuTerminateMotionBuffer earlier or avoid sending it at all.
