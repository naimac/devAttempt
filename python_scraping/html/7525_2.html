%1: EtherCAT disconnected.

Parameter:

%1......

Configuration path or name of object

Message Class:

3 -Error Minor

Reason:

The EtherCAT slave has no ethernet link.

Effect:

EtherCAT communication is not possible.

Solution:

Check EtherCAT cabling and connect device with ethernet. An established ethernet link is indicated by the link LED.
