%1: Module not found in projection.

Parameter:

%1......

Configuration path or name of object

Message Class:

3 -Error Minor

Reason:

The configured KBUS module could not be found in the engineering data file (ENI).

Effect:

Operation of the KBUS module is not possible.

Solution:

Adapt the configuration file (.cfg) or the engineering data file (ENI). Check if the address of the KBUS-Modules (hex switch) matches the values in ENI/cfg.
