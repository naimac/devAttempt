Messages

4000_3

Interface %1 of version %2 is not compatible with the installed Mc version (%3)

Parameter:

%1......

Interface identifier

%2......

Mc version where the used interface stems from

%3......

Mc version that is installed on the system

Message Class:

4 -Warning

Reason:

The Mc interface version that is used is not compatible with the installed Mc version.

Effect:

No application program can be started.

Solution:

Provide information to your distributor. Match the Mc-Interface and the installed Mc-Version.
