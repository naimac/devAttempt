Messages

4000_555

Kinematic %3 (ID %4) is singular at %1 %2

Parameter:

%1......

Position name

%2......

Position ID

%3......

Kinematic name

%4......

Kinematic ID

Message Class:

3 -Error Minor

Reason:

The kinematic cannot move to the specified position, because this leads to a singularity.

Effect:

Robot and motion program are stopped.

Solution:

Modify path to avoid singularity.
