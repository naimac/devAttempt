Messages

4000_303

Kinematic %1: inverse transformation only possible if forward transformation = ON

Parameter:

%1......

Kinematic name

Message Class:

4 -Warning

Reason:

Configuration error: If an inverse transformation is activated, then also the forward transformation must be activated.

Effect:

Kinematic is not available.

Solution:

Check the configuration data. Activate forward transformation.
