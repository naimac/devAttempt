Messages

56_3013

Not all axes of robot '%1' are simulated.

Parameter:

%1......

Robot name

Message Class:

8 -

Reason:

The robot has simulated as well as non-simulated axes configured.

Effect:

Robot movement may result in unexpected directions.

Solution:

Use either simulated axes only, or use the robot only with non-simulated axes.
