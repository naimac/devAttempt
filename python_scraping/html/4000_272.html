Messages

4000_272

Programmed deceleration %5 for joint %4 (comp. %3) must be greater than 0. Dynamics %1 (addr. %2) invalid.

Parameter:

%1......

Dynamics name

%2......

Dynamics ID

%3......

Component name

%4......

Joint number of component

%5......

Invalid value

Message Class:

4 -Warning

Reason:

The application program tried to set the programmed deceleration to a value near zero. This is not allowed.

Effect:

Dynamics will not be changed. The motion program will be stopped.

Solution:

Check application or dynamics data in user program.
