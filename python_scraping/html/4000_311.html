Messages

4000_311

Kinematic %1: PrewarnTime (%2 s) higher than LookAheadTime (%3 s). Setting PrewarnTime to %3 s.

Parameter:

%1......

Kinematic name

%2......

Desired prewarn time

%3......

Configured prewarn time

Message Class:

3 -Error Minor

Reason:

Configuration error: For prewarn-jobs an excessive prewarn-time has been configured. The prewarn time must not be greater than the configured look-ahead time.

Effect:

Trigger with prewarn time do not work correctly.

Solution:

Check configuration data.
