Messages

2236_53

%1: Drive not ready to operate (SERCOS drive state 0x%2).

Parameter:

%1......

Hardware configuration path or name of drive

%2......

Value of sercos stateword S-0-0135

Message Class:

3 -Error Minor

Reason:

In drivecom state 'operation enabled' drive signalled in sercos stateword S-0-0135 via bits 14 and 15, that sercos state 'in operation, with torque' is not active.

Effect:

Drive is set to drivecom state 'fault'.

Solution:

Eliminate error reason - it could be caused by the controller (e.g. invalid set values) or by the drive environment (e.g. power supply).
