%1: '%2': CycleTime too small (act = %3 ns, min = %4 ns, section = '%5'/'%6').

Parameter:

%1......

Configuration path or name of object

%2......

ENI filename

%3......

Actual number

%4......

Minimum number

%5......

Section of node

%6......

Subsection of node

Message Class:

3 -Error Minor

Reason:

The cycle time configured in the ENI file is too small.

Effect:

The operation of the EtherCAT segment is not possible.

Solution:

Use a higher cycle time.
