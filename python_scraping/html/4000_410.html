Messages

4000_410

Kinematic %1 (Joint %2): Setting position not possible. Drive is not referenced.

Parameter:

%1......

Kinematic name

%2......

Joint name

Message Class:

4 -Warning

Reason:

The position cannot be set when the actual position of the drive is not available. The drive signals 'reference lost' by its status bit.

Effect:

The joint's position has not been set.

Solution:

Before setting the position of a joint the drive must be referenced or the status bit 'reference lost' has to be cleared.
