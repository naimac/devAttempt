4000_57

The system's MCU heap is depleted. (configured size: %1 kB)

Parameter:

%1......

Configured size of the MCU heap

Message Class:

3 -Error Minor

Reason:

Not enough memory for Motion Control Unit. Perhaps there is an error in the application programm (objects are not deleted).

Effect:

No more motion programs can be started or continued.

Solution:

Increase the configured size of the motion database size or check the application program.
