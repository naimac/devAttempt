Messages

2236_226

%1: Duplicate SERCOS-address (%2) detected on bus.

Parameter:

%1......

configuration path or name of bus

%2......

SERCOS-address that exists more than once on the bus

Message Class:

3 -Error Minor

Reason:

Several SERCOS-devices on the bus are using the same SERCOS-address.

Effect:

CP 1 cannot be reached. Initialization of bus is retried periodically

Solution:

Check SERCOS device addresses.
