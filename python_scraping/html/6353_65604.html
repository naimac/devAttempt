Messages

6353_65604

%1: Init command '%2' failed during state transition '%3' (AL Status Code = 0x0044 Mailbox FoE).

Parameter:

%1......

Configuration path or name of object

%2......

Init command

%3......

State transition

Message Class:

3 -Error Minor

Reason:

An init command failed during a state transition of the EtherCAT slave.

Effect:

The operation of the EtherCAT segment is restricted. The operation of the effected EtherCAT slave is not possible.

Solution:

Check the failed init command.
