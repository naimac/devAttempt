Messages

4000_181

Motion command on moving object is too short: %1 %2 (ID %3).

Parameter:

%1......

Length of the segment

%2......

Segment name

%3......

Segment ID

Message Class:

5 -Info

Reason:

A change from a fixed reference system to a moving reference system was programmed with a very short movement.

Effect:

Excessive dynamics may occur, possibly leading to drive failure.

Solution:

Make the movement from a fixed reference system to a moving reference system longer.
