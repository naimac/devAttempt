Messages

6983_10

%1: %2-PDO memory data mapping for bytes %3..%4 (%5) not found or incorrect.

Parameter:

%1......

Configuration path or name of object

%2......

Value "RX" indicates output data from control view, value "TX" indicates input data from control view

%3......

First byte in the memory area of the module that has been searched for

%4......

Last byte in the memory area of the module that has been searched for

%5......

Text describing the kind of data in the mapping (e. g. DI, DO, DOSTAT, AI...)

Message Class:

3 -Error Minor

Reason:

The described bytes have not been found in the PDO mapping in the engineering data file (ENI) or the bytes are not packed (a single sequence).

Effect:

Operation of the object is not possible.

Solution:

Adapt the configuration file (cfg) or the the engineering data file (ENI).
