Messages

56_1140

'%1': no Function Block has been called in the last PLC cycle (PLCopen State: '%2').

Parameter:

%1......

name of the axis

%2......

PLCopen state

Message Class:

3 -Error Minor

Reason:

No Function Block has been called in the last PLC cycle. Therefore the axis has not recevied new set values. As long as the axis is moving at least on motion function block has to be called on the axis each PLC cycle.

Effect:

The axis switches to the PLCopen state 'ErrorStop'. The drive is disabled.

Solution:

The active FB of the axis has to be called each PLC cycle.
