Messages

6353_13

%1: Working counter of cyclic command %2 is invalid (expected = %3, received = %4).

Parameter:

%1......

Configuration path or name of object

%2......

Number of cyclic command

%3......

Expected working count

%4......

Received working count

Message Class:

4 -Warning

Reason:

At least one slave has not processed the cyclic command as expected.

Effect:

The process data of the cyclic command is not processed.

Solution:

If the working counter continues to return with a wrong value, then at least one defect slave must be replaced in the EtherCAT segment. Otherwise an adequate message is dropped.
