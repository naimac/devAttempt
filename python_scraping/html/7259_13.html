Messages

7259_13

%1: Error in %2 endpoint config: fast cycletime (%3 s) slower than slow cycletime (%4 s).

Parameter:

%1......

Configuration path or name of object

%2......

Type of endpoint

%3......

Cycletime of fast endpoint in seconds

%4......

Cycletime of slow endpoint in seconds

Message Class:

3 -Error Minor

Reason:

The configured cycletime of the fast endpoints is bigger than the configured cycletime of the slow endpoints.

Effect:

No update is executed due to invalid configuration.

Solution:

The cycletime of the fast endpoints has to be configured smaller or equal to the cycle time of the slow endpoints.
