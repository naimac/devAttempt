Messages

2236_14

%1: Invalid value at read parameter: 0x%2, element %3, value %4.

Parameter:

%1......

Hardware configuration path or name of device

%2......

Parameter number

%3......

Element number

%4......

Read value

Message Class:

3 -Error Minor

Reason:

An invalid value was received at reading a parameter.

Effect:

The parameter transfer (read or write) can't be finished successfully.

Solution:

Check if the used device behaves as defined in the sercos standard.
