Messages

4000_182

Excessive orientation change relative to circular plane of %1 deg (max. %2 deg) on segment %3 (ID %4).

Parameter:

%1......

Programmed orientation change with respect to circular plane

%2......

Max. orientation change with respect to circular plane

%3......

Segment name

%4......

Segment ID

Message Class:

3 -Error Minor

Reason:

The maximum orientation change with respect to the circular plane has been exceeded.

Effect:

Circ-segment will not be accepted.

Solution:

Choose different orientation for start and/or end position such that there is less orientation change with respect to the circular plane, or use different kind of orientation interpolation.
