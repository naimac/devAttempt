Messages

6353_55

%1: Timing violation occurred (%2).

Parameter:

%1......

Configuration path or name of object

%2......

Error cause (possible values: 'DMA', 'SW normal', 'SW early')

Message Class:

3 -Error Minor

Reason:

A timing violation occurred.

Effect:

The projected frames can not be transferred with the calculated timing.

Solution:

According to the error cause the folllowing configuration entries must be increased: 'DMA': dmaTime; 'SW normal': startOffset; 'SW early': earlyStartOffset. For an optimum configuration please read the relevant chapter in the system manual.
