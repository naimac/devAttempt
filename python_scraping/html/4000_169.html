Trying to create a circular segment which is variable

Parameter:

Message Class:

4 -Warning

Reason:

It has been tried to create a circular segment, whose start position, help psoition, and end position have different reference systems. At least one of these reference systems is moving.

Effect:

The segment cannot be created.

Solution:

Use identical reference system for start position, help position, and end position of the circle.
