Messages

4000_561

Invalid tool for inverse transformation of %1 (ID %2) by kinematic %3 (ID %4)

Parameter:

%1......

Object name

%2......

Object ID

%3......

Kinematic name

%4......

Kinematic ID

Message Class:

3 -Error Minor

Reason:

An invalid tool has been used for the inverse transformation of the kinematic. Some robots only allow tools with less than 6 degrees of freedom, e.g. straight tools.

Effect:

Robot cannot perform Cartesian movements.

Solution:

Use valid tool.
