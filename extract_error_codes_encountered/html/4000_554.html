Messages

4000_554

Wrist joints of kinematic %3 (ID %4) are singular at %1 (ID %2)

Parameter:

%1......

Position name

%2......

Position ID

%3......

Kinematic name

%4......

Kinematic ID

Message Class:

3 -Error Minor

Reason:

The kinematic cannot move to the specified position, because this leads to a wrist joint singularity.

Effect:

Excessive joint dynamics and path deviation may occur.

Solution:

Modify path to avoid singularity.
