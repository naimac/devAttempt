Messages

2236_171

%1: Drive not possible to switch on, power not applied (SERCOS drive state 0x%2).

Parameter:

%1......

Hardware configuration path or name of drive

%2......

Value of sercos stateword S-0-0135

Message Class:

4 -Warning

Reason:

Drivecom state transition 3 'Switch on' is not possible, because drive signals in stateword S-0-0135 via bit 15, that power is not applied.

Effect:

Drive is set to drivecom state 'fault'.

Solution:

Check drive and its power supply.
