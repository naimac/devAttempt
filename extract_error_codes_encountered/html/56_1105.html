%1: Drive is not ready to be enabled

Parameter:

%1......

name of the axis

Message Class:

3 -Error Minor

Reason:

While enabling the drive it was detected that enabling is prevented. There may be several reasons depending on the type of drive and the assembly: Power is not applied to the drive, missing enable signal, ... .

Effect:

The drive can not be enabled. If the error occured while homing, the axis switches to the state 'ErrorStop'. If the error occured due to calling a FB MC_Power with Enable=TRUE the axis stays in the PLCopen state 'Disabled'. If the next time the FB MC_Power is called with Enable=TRUE, the FB will be rejected with an error.

Solution:

The error cause has to be cleared: Power has to be applied to the drive, enable signal has to be set, ... .
