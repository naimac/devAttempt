4000_767

Inertia parameters for tool %1 are physically inconsistent (%2-direction)

Parameter:

%1......

Tool name

%2......

Coordinate direction

Message Class:

4 -Warning

Reason:

After transformation of inertia moments from the original coordinate system (cog) to the tool system (using parallel axis theorem) the resulting inertia moments are negative.

Effect:

Calculated torques will not be physical.

Solution:

Review the following parameters of the tool: mass, distanceCenterGravityX, distanceCenterGravityY, distanceCenterGravityZ, inertiaMomentJxx, inertiaMomentJyy, inertiaMomentJzz, and inertiaMomentAroundCG
