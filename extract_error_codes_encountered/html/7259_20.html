Messages

7259_20

%1: Line frequency error

Parameter:

%1......

Configuration path or name of object

Message Class:

4 -Warning

Reason:

The detected line frequency does not match the configured line frequency

Effect:

Input values that depend on the line frequency may not be correct and output values that depend on the line frequency may not be set in the way it is expected.

Solution:

Configure the correct line frequency and/or check the line frequency of connected inputs.
