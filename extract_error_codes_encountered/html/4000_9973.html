Messages

4000_9973

Velocity of cartesian position exceeds limits even if stopped on path at %1

Parameter:

%1......

segment position

Message Class:

3 -Error Minor

Reason:

The cartesian velocity limit is exceeded even if stopped on path at the given path position. This happens due to tracking.

Effect:

It is not possible to pass the given path position without exceeding the velocity limits.

Solution:

Increase the velocity limit or reduce the tracking velocity.
