Messages

2220_12018

%1: Value '%2' for configuration entry '%3' is out of range [%4..%5]. Default value '%6' will be used.

Parameter:

%1......

configuration path or name of the device

%2......

the invalid value configured for the configuration entry

%3......

name of configuration entry

%4......

the smallest valid value

%5......

the largest valid value

%6......

default value for configuration entry

Message Class:

4 -Warning

Reason:

An invalid value has been configured for an configuration entry.

Effect:

The default value will be used instead of the configured value.

Solution:

To prevent this message a valid value has to be configured for this configuration entry.
