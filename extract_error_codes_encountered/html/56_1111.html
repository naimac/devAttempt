Messages

56_1111

'%1': software position limit (%2) exceeded (%3)

Parameter:

%1......

name of the axis

%2......

software position limit

%3......

(set) position of the axis

Message Class:

3 -Error Minor

Reason:

A software position limit has been exceeded.

Effect:

The drive is disabled and the axis switches to 'ErrorStop'.

Solution:

If the axis is standing outside the software position limits, the axis can only be moved towards the software position limits.
