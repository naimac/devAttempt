%1: Ethernet frame was received too late.

Parameter:

%1......

Configuration path or name of object

Message Class:

3 -Error Minor

Reason:

An Ethernet Frame was received too late, because either the EtherCAT segment is too big for the configured cycle time or the CPU power is not sufficient for the configured cycle time.

Effect:

The operation of the EtherCAT segment is not possible.

Solution:

Reduce the cycle time or the number of slaves in the EtherCAT segment, if the EtherCAT segment is very big. Use a more powerful CPU, if the EtherCAT segment is small.
