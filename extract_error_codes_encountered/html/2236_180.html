2236_180

%1: Duplicate SERCOS-address detected on bus.

Parameter:

%1......

Configuration path or name of bus

Message Class:

3 -Error Minor

Reason:

Several SERCOS-devices on the bus are using the same SERCOS-address.

Effect:

CP 1 cannot be reached. Initialization of bus is retried via QuitError-Task.

Solution:

Check SERCOS device addresses
