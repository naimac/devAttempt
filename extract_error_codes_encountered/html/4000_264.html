Messages

4000_264

Programmed cart. velocity %4 (component %3) must be greater than 0. Dynamics %1 (addr. %2) invalid.

Parameter:

%1......

Dynamics name

%2......

Dynamics ID

%3......

Component name

%4......

Invalid value

Message Class:

4 -Warning

Reason:

The application program tried to set the cartesian velocity to a value near zero. This is not allowed.

Effect:

The command for changing the velocity value is not executed and an error occurs. If the application continues executing the program further errors will occur.

Solution:

Change value for cartesian velocity in the application program.
